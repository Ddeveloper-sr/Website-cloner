## 🚀 How to Use (Super Simple)

### ✅ Option 1: Use the Live Demo (Recommended - No Download!)
This is the fastest way for most people.

1. Open the **Live Demo**: [https://website-cloner-orcin.vercel.app](https://website-cloner-orcin.vercel.app)
2. Enter the full URL of the website you want to clone (e.g. `https://example.com`)
3. Choose your **Mode**:
   - **Clone Mode** — Best for full websites (recommended)
   - **Component Mode** — Extract just one section (navbar, hero, footer, button, etc.)
   - **Compare Mode** — Clone two websites and compare size/pages/assets side-by-side
4. (Optional) Adjust settings:
   - Max pages / depth
   - Choose a CORS proxy (try different ones if one fails)
   - Enable sitemap crawling for better results
5. Click **Start** and wait while it downloads everything.
6. When finished, click **Download ZIP**.
7. Extract the ZIP and open `index.html` inside — everything works offline!
